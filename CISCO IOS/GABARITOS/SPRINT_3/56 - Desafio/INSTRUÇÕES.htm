Você foi contratado para configurar a conexão entre duas redes distintas da empresa Ganondorf .Inc.

A empresa tem duas filiais, uma em Minas Gerais e outra no Paraná.

 Cada filial tem seu esquema de VLAN, seguindo as tabelas abaixo:

REDE - MG

VLAN NOME REDE MÁSCARA
10 USUARIOS 172.16.0.0 /24
50 SERVIDORES 172.16.1.0 /24
99 NATIVA - -

REDE - PR

VLAN NOME REDE MÁSCARA
10 USUARIOS 172.16.2.0 /24
50 SERVIDORES 172.16.3.0 /24
99 NATIVA - -

 Você deve seguir as seguinte tabela de roteamento:

REDE - MG

NOME IP VLAN
PC-01 5º IP VÁLIDO DA REDE 10
PC-02 10º IP VÁLIDO DA REDE 10
SRV-01 2º VÁLIDO DA REDE 50

O roteador deve ser configurado utilizando o método de Router-on-a-Stick, para fornecer roteamento para todas as VLANs, utilizando a seguinte tabela

NOME SUBINTERFACE VLAN IP
RT-01 g0/0.10 10 172.16.0.1
RT-01 g0/0.50 50 172.16.1.1

REDE - MG

NOME IP VLAN
PC-03 5º IP VÁLIDO DA REDE 10
PC-04 10º IP VÁLIDO DA REDE 10
SRV-02 2º VÁLIDO DA REDE 50

O roteador deve ser configurado utilizando o método de Router-on-a-Stick, para fornecer roteamento para todas as VLANs, utilizando a seguinte tabela

NOME SUBINTERFACE VLAN IP
RT-02 g0/0.10 10 172.16.2.1
RT-02 g0/0.50 50 172.16.3.1

 Os dois roteadores estão se comunicando utilizando uma conexão WAN Serial, seguindo o seguinte endereçamento:

REDE WAN - 200.200.100.0/30

NOME INTERFACE IP
RT-01 s0/0/0 1º IP VÁLIDO DA REDE
RT-02 s0/0/0 2º IP VÁLIDO DA REDE
