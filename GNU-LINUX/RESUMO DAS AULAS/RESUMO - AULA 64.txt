Editores de Texto para Terminal
    nano
        Pesquisar: Ctrl+W
        Copiar = Alt+6
        Recortar = Ctrl+K
        Colar = Ctrl+U

        Salvar = Ctrl+O
        Sair = Ctrl+X

        Desfazer = Alt+U

    vim (Versão melhorada do 'vi')
        vim.tiny - Versão menor do vim, onde alguns recursos não estão disponíveis

        : = Inserir comando no Vim
        i = Entrar no modo de Inserção
        :w = Salvar o arquivo
        :q = Sair do arquivo
        v = Entrar no modo visual
        y = Copiar
        p = Colar
        u = Desfazer

        .vimrc = Arquivo de configuração do vim
            set number

    emacs

Arquivos de configuração do Linux
    /etc/hostname - Armazena o nome do computador

Simbolos utilizados no Linux
    ^ = Ctrl
        ^G = Ctrl + G
    M = Alt

Comandos complementares

    > = Insere o conteúdo de um comando para um arquivo de texto
        Exemplo: tail arquivo.txt > ultimas-10-linhas.txt

Pesquisar
    Diferença entre usar o '>' e '>>'?
