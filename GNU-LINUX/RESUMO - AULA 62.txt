adduser - Adiciona um usuário ao Linux
deluser - Deleta um usuário do Linux
addgroup - Adiciona um grupo ao Linux
delgroup - Deleta um grupo do Linux
passwd - Trocar a senha de um usuário

adduser [usuario] [grupo] - Adiciona um usuário a um grupo
deluser [usuario] [grupo] - Retirar um usuário de um grupo

/etc/passwd - Arquivo que contém os usuários do Linux
/etc/group - Arquivo que contém os grupos do Linux
/etc/shadow - Arquivo que contém as senhas dos usuários

UID - Identificação numérica única do Usuário
GID - Identificação numérica única do Grupo

/var/log/syslog - Arquivo de armazenamento dos Logs do Sistema
/var/log/auth.log - Arquivo de armazenamento de Autenticação dos Usuários

tail - Exibe as últimas 10 linhas de um Arquivo
tail -f - Exibe as últimas linhas de um arquivo em tempo real
grep - Filtrar informações específicas de um arquivo (quando usado com o |)
